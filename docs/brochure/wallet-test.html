Phase 3 — P3-1

BRC-100 wallet primitive, live.

This page exercises the bMovies BRC-100 wallet primitive end-to-end: it probes your machine for a BSV Desktop / Metanet Client instance, asks it for a public key (triggering the wallet's permission prompt on first contact), and can send a small payment of your choice from your wallet to any address via createAction. bMovies never touches the private key — the wallet signs and broadcasts.

No wallet? Scroll down for the install prompt. bMovies is a BRC-100 app and targets the Metanet App Catalog, not phone stores.
